## دليل سكواد باسترز للمبتدئين

دليل ويب تفاعلي باللغة العربية يعرّف اللاعبين الجدد على أهم شخصيات لعبة **Squad Busters**، مع صفحة رئيسية قابلة للبحث وصفحات تفصيلية لكل بطل تتضمن المهارات، النصائح، ومعرض صور توضيحي عالي الدقة (SVG).

### المتطلبات

- Node.js 18 أو أحدث
- npm

### التثبيت والتشغيل المحلي

```bash
npm install
npm run generate:art   # خطوة اختيارية إذا احتجت إعادة إنشاء الرسومات
npm run dev
```

يفتح الخادم على العنوان `http://localhost:3000`.

### سكريبتات npm

- `npm run dev` — تشغيل الخادم التطويري.
- `npm run build` — إنشاء نسخة الإنتاج.
- `npm run start` — تشغيل نسخة الإنتاج.
- `npm run lint` — تشغيل ESLint.
- `npm run generate:art` — إعادة إنشاء رسومات الشخصيات (SVG) داخل `public/images/characters`.

### هيكل المشروع

```
app/
  layout.tsx           # تخطيط الموقع العام
  page.tsx             # واجهة عرض الشخصيات
  characters/[slug]/   # صفحات تفاصيل الأبطال
components/            # مكوّنات واجهة القوائم والبطاقات
data/characters.ts     # قاعدة بيانات الشخصيات (يمكن توسيعها بسهولة)
public/images/         # الرسومات التوضيحية المنشأة تلقائيًا
scripts/generate-art.js# سكريبت إنشاء الرسومات
```

### التخصيص والتوسعة

- لإضافة شخصية جديدة: حدّث ملف `data/characters.ts` ثم شغّل `npm run generate:art` لإضافة رسمها.
- التصميم مبني على Next.js (App Router) ويعتمد على CSS مخصص قابل للتعديل بسهولة.

### المصادر

- تم بناء الرسومات بالاعتماد على لوحة الألوان المطروحة في **Supercell fankit**، مع الحفاظ على الطابع الأصلي للشخصيات.
- الموقع متوافق مع أجهزة الهواتف، الأجهزة اللوحية، وأجهزة سطح المكتب، ومهيأ للنشر عبر Vercel.
